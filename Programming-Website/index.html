Looking for some more advanced projects for your resume?

Try some of our most popular projects for more experienced programmers.

The Voiceinator

Build a Javascript app that reads out what you type in the text box. Have fun with this app by controlling the pitch and rate of the words being spoken!

The Locater

Build a Javascript app where the user can search and filter for cities and states around the globe! The location plus the population is displayed.

The Link Finder

This handy Javascript app can be used on any webpage or application to filter for all the links on a page. The links are highlighted in a white color.

The Gallery

This beautiful Javascript application displays a gallery of images using Flicker's API. These photos are displayed using a smooth transition effect.
